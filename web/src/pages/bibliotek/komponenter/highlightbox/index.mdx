# HighlightBox

<MDXCustom.Preamble>
  HighlightBox [HighlightBox] hjelper innbygger å oppfatte viktigere innhold, der hvor det er behov for å utheve et innholdsområde fremfor
  annet innhold på samme side
</MDXCustom.Preamble>
