# Tooltip

<MDXCustom.Preamble>
  Tooltip [Tooltip] hjelper innbygger å forstå et ord eller en setning bedre ved å vise en forklaring i en HelpBubble når teksten hovres
  eller klikkes på.

  Ved bruk av flere tooltips på en side så skal de wrappes i TooltipOpenProvider som eksporteres fra Tooltip.
</MDXCustom.Preamble>

## Usage
```javascript
import React from 'react';

import Tooltip, { TooltipOpenProvider } from '../Tooltip';

  return (
    <TooltipOpenProvider>
      <>
        <Tooltip description={'Dette er tekst som skal fylle HelpBubble'} >{'ordet'}</Tooltip>
        {'Dette er '}
        <Tooltip description={'Dette er tekst som skal fylle HelpBubble'} >{'enda et tooltip ord'}</Tooltip>
      </>
    </TooltipOpenProvider>
  );
```
